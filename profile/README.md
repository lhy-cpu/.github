## 🧭 学习历程与技术路线

从上层抽象 ➡️ 底层硬件 ➡️ 人工智能前沿，构建全生命周期开发能力。

<br>

### ☕ 第一阶段：应用基石 (Java 生态)
📌 **核心目标**：奠定扎实的面向对象编程思维
- 💻 **软件开发**：构建跨平台 Java 桌面端与 JavaWeb 后端服务
- 📱 **移动开发**：编写原生 Android 移动端应用
- 🎮 **游戏模组**：开发高度自定义的 Minecraft (Java版) 插件

<div align="center">⏬</div>

### ⚙️ 第二阶段：深入底层 (C/C++ & 嵌入式硬件)
📌 **核心目标**：探究“计算机底层是如何工作的”
- 🔧 **主控芯片**：STC单片机、STM32、ESP 系列、Infineon AURIX™ TC 车规芯片
- 💻 **开发板与系统**：树莓派 (Raspberry Pi)、NVIDIA Jetson、RK(瑞芯微) 系列、嵌入式 Linux 开发
- 🔌 **硬件设计 (PCB)**：2-6层板设计、中高速 USB2.0、BGA 走线、独立设计 DC-DC / 飞控 / FOC 控制板
- 🔮 **前沿探索**：积极研究 FPGA 架构与硬件描述语言

<div align="center">⏬</div>

### 🌐 第三阶段：Web 生态与全栈
📌 **核心目标**：掌控现代产品完整的生命周期
- 🎨 **前端开发**：运用 HTML/CSS/JavaScript 构建交互式 Web 原生页面
- 🚀 **后端服务**：搭建异步 Node.js 与轻量级 Flask 后端服务
- 🧩 **辅助增强**：编写提升生产力的浏览器插件与油猴脚本

<div align="center">⏬</div>

### 🧠 第四阶段：智能与自动化 (Python & AI)
📌 **核心目标**：赋能自动化并拥抱次世代人工智能
- 🤖 **语言模型**：探索大语言模型 (LLM) 推理与集成应用
- 👁️ **计算机视觉**：深耕 AI 视觉与传统计算机视觉 (CV)
- ⚙️ **自动化工具**：编写实用的爬虫小工具和自动化脚本
